Document Tags
Tag Name Purpose
html HTML tag makes alterations and settings to the whole document
head Head tag The head element is a container for metadata (data about data) and is placed in between html and body tags. So it's where you set the title, seo stuff, other information.
body Body Tag Containst he content of the HTML page
title Title tag Sets the title of the tag, which shows up in the tab of a web browser and is used in web searches.
meta Meta Had to google this one: "Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata. The metadata can be used by browsers (how to display content or reload page), search engines (keywords), or other web services."
span Span tag Can isolate a specific bit of text without making any viewable changes.
blockquote Blockquote Puts whatever is in the blockquote in the middle of the margin, typically with italicized font and larger than normal font, and a different bakcground color. Used most of the times for accenting that text
br Break Breaks the text from goipng on one line
to going down one line
